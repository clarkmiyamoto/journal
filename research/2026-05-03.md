# Augmented Marginal OT-SCSI: Recovering Optimal Convergence in Lifted Spaces

## 1. Background & The Convergence Bottleneck
In the problem of Generative Modeling from Black-box Corruptions, we aim to recover a true data distribution $\pi = \mathcal N(\mu, \Sigma)$ from corrupted observations $Y = \mathcal F(X) \sim \mathcal N(\mu, \Sigma + \sigma^2 \mathbb I)$. The original **Self-Consistent Stochastic Interpolant (SCSI)** algorithm achieves a fast, squared-error contraction rate corresponding to $\epsilon=0$ by learning a deterministic Marginal Optimal Transport (OT) map.

When attempting to handle dimensional mismatches (where $Y \in \mathbb{R}^m$ and $X \in \mathbb{R}^n$, $m < n$), one intuitive approach is the **Lifted Conditional SCSI**. Here, we simulate an interpolant $I_t^{(k)} | Y^{(k)}=y \overset{d}{=} (1-t) Z+ t X^{(k)}$ using independent noise $Z \sim \mathcal N(0, \mathbb I)$ and learn the conditional drift $b_t(x|y)$. 

**The Bottleneck:** The lifted conditional algorithm converges slower ($\epsilon=1$ rate). By regressing on the conditional expectation, the network learns the Wiener filter matrix:
$$B_k^{Lifted} = \Sigma_k(\Sigma_k + \sigma^2 \mathbb I)^{-1}$$
Because it shrinks the variance too much, the ODE must compensate by injecting the independent noise $Z$, which actively dilutes the true signal from $Y$. Conversely, the optimal Marginal OT map uses the Bures-Wasserstein square root matrix $B_k^{ODE} = (\Sigma_k(\Sigma_k + \sigma^2 \mathbb I)^{-1})^{1/2}$, extracting strictly more signal and avoiding the noise dilution.

## 2. The Dimensionality Challenge
If $m < n$, we cannot build a purely deterministic OT map from $Y$ to $X$, because $f(Y)$ would only ever span an $m$-dimensional manifold. To recover the full distribution, we *must* inject $n-m$ dimensions of independent noise. Reflowing the conditional algorithm fails because the observable dimensions are already "straight," causing the update to collapse into the identity map.

## 3. The Solution: Augmented Marginal OT-SCSI
To regain the $\epsilon=0$ speedup along the observable dimensions while properly handling the unobservable null-space, we switch from *Conditional* Flow Matching to an **Augmented Marginal** Flow Matching procedure.

1. **Augmentation (Padding):** Project the observation $Y$ into the full $\mathbb{R}^n$ space using the pseudo-inverse of the forward operator, and fill the unobservable null-space with independent noise.
2. **Minibatch Optimal Transport:** Abandon the conditional pairing. Instead, treat the augmented observations $\tilde{Y}$ and current generations $X^{(k)}$ as two separate marginal pools, and pair them using a discrete Optimal Transport solver (e.g., Sinkhorn).
3. **Marginal Flow Matching:** Train a standard, unconditional Flow Matching ODE on these new optimal pairs. 

This forces the network to learn the Bures-Wasserstein $B_k^{ODE}$ mapping in the observable subspace (recovering the fast $\epsilon=0$ rate), while acting as optimal noise replacement in the unobservable dimensions.

## 4. Pseudocode
```pseudo
\begin{algorithm} 
\caption{Augmented Marginal OT-SCSI} 
\begin{algorithmic}
\REQUIRE Corrupted dataset $\mathcal{D}_Y = \{y_1, \dots, y_N\} \subset \mathbb{R}^m$ \REQUIRE Forward operator $H \in \mathbb{R}^{m \times n}$ (such that $y = Hx + \text{noise}$) 
\REQUIRE Base distribution $\mathcal{N}(0, I_n)$ 
\REQUIRE Initial model for drift $b_t^{(0)}(x)$ 
\STATE Compute $H^\dagger = H^T(HH^T)^{-1}$ \COMMENT{Pseudo-inverse} \STATE Compute $U_\perp \in \mathbb{R}^{n \times (n-m)}$ \COMMENT{Orthonormal basis for null-space of $H$} \FOR{$k = 0$ \TO $K-1$} \STATE \textbf{// 1. Generate current samples} \FOR{$i = 1$ \TO $M$ (minibatch size)} \STATE Sample $X_{i,0} \sim \mathcal{N}(0, I_n)$ \STATE Solve $\frac{dX_{i,t}}{dt} = b_t^{(k)}(X_{i,t})$ from $t=0$ to $t=1$ \STATE Store $X_i^{(k)} = X_{i,1}$ \ENDFOR \STATE \textbf{// 2. Augment observations to full dimension $\mathbb{R}^n$} \FOR{each $y_j$ in a minibatch from $\mathcal{D}_Y$} \STATE Sample $Z_j \sim \mathcal{N}(0, I_{n-m})$ \COMMENT{Independent noise for null-space} \STATE Compute $\tilde{y}_j = H^\dagger y_j + U_\perp Z_j$ \ENDFOR \STATE Let $\tilde{\mathcal{D}}_Y$ be the batch of augmented observations $\{\tilde{y}_j\}$. \STATE \textbf{// 3. Minibatch Optimal Transport Pairing} \STATE Compute cost matrix $C_{i,j} = \| X_i^{(k)} - \tilde{y}_j \|^2$ \STATE Solve discrete OT (e.g., via Sinkhorn algorithm) to find optimal pairs $(X_i^{(k)}, \tilde{y}_{\pi(i)})$ \STATE \textbf{// 4. Train Marginal Flow Matching} \STATE Sample time steps $t \sim \mathcal{U}[0, 1]$ \STATE Construct interpolant: $I_{t,i} = (1-t) \tilde{y}_{\pi(i)} + t X_i^{(k)}$ \STATE Target velocity: $V_{t,i} = X_i^{(k)} - \tilde{y}_{\pi(i)}$ \STATE Update drift $b_t^{(k+1)}(x)$ by performing gradient descent steps to minimize: \[ \mathcal{L} = \mathbb{E}_{t, i} \left[ \| b_t^{(k+1)}(I_{t,i}) - V_{t,i} \|^2 \right] \] \ENDFOR \ENSURE Final generative model ODE defined by $b_t^{(K)}(x)$ \end{algorithmic} 
\end{algorithm}
```
